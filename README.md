# MNIST Mixture Models Analysis

An unsupervised machine learning project exploring how different mathematical distributions like **Gaussian, Laplacian, and Bernoulli** cluster and generate handwritten digits from the MNIST dataset. 

This repository implements Expectation-Maximization (EM) algorithms to evaluate how well each distribution models the underlying density of the data, using both continuous (PCA-compressed) and discrete (binarized) feature spaces.

## Project Overview

This project compares three distinct distributions by evaluating two core capabilities:
1. **Clustering Performance:** How accurately can the model group unseen handwriting styles into clusters without ground-truth labels?
2. **Generative Capabilities:** How realistic are the new digits generated by sampling from the learned probability distributions?

---

## Repository Structure

Based on the pipeline, the project is divided into five sequential Jupyter Notebooks:

* **`1_Preprocessing.ipynb`**
  Loads the full 70,000-image MNIST dataset. Creates two distinct data pipelines:
  * **PCA Track:** Compresses data to 50 dimensions (continuous features) for the Gaussian and Laplacian models.
  * **Binary Track:** Thresholds the raw 784 pixels into strict 0s and 1s (discrete features) for the Bernoulli model.
  
* **`2_Gaussian_MM.ipynb`**
  Implements `scikit-learn`'s Gaussian Mixture Model on the PCA data. 
  
* **`3_Laplacian_MM.ipynb`**
  Contains the implementation for the Laplacian distribution MM.
    
* **`4_Bernoulli_MM.ipynb`**
  Contains the implementation for the Bernoulli distribution MM. Operates without PCA, modelling the 784-dimensions as a binary pixel space.
  
* **`5_Final_comparison.ipynb`**
  Loads the saved metrics and generated samples to provide a side-by-side mathematical and visual analysis of the models.
